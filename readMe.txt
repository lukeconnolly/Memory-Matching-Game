To run: Change directory to assignment 2 file and install the mime types module. To do this, type 
"npm install mime-types". Then type "node server.js" to run. Go to "localhost:2406"
in a web browser, then type your name in the prompt and submit it. After this is done, you can start
clicking cards and playing the game. The size of the game board will increase upon winning a game,
with the size capped at 10x10.



Developed under: Windows 10

Tested with: Google Chrome, Microsoft Edge

Files: Assignment2(folder) : server.js, readMe.txt, makeBoard.js, public_html(folder)
       public_html(folder) : 404.html, gameLogic.js, index.html, favicon.ico, assignment2css.css
